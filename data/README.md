# Data Directory

Place your `creditcard.csv` dataset in this folder.

## Download Dataset

1. Visit: https://www.kaggle.com/mlg-ulb/creditcardfraud
2. Download `creditcard.csv`
3. Place it in this directory

The dataset contains:
- 284,807 transactions
- 30 features (Time, Amount, V1-V28)
- Binary class label (0 = Normal, 1 = Fraud)
- Size: ~150 MB

**Note**: This folder is ignored by git (.gitignore) to avoid committing large data files.
